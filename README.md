wa# SecureAuto Insurance Website

A modern, responsive insurance website built with HTML, CSS, and JavaScript. Includes:

- Project structure:

```
project-root/
├── index.html
├── quote.html
├── about.html
├── faq.html
├── landing.html
├── css/
│   └── styles.css
├── js/
│   ├── main.js
│   └── quote.js
└── README.md
```
- Modern CSS styling (css/styles.css)
- Interactive quote calculation (js/main.js)
- Quote-specific logic placeholder (js/quote.js)

## Features
- Auto insurance option with quote form
- Navigation between all pages
- Responsive, clean design

## Usage
Open `landing.html` in your browser to start. Navigate using the menu or buttons.

## Customization
- Update content in HTML files as needed
- Adjust styles in `css/styles.css`
- Modify logic in `js/main.js` or `js/quote.js`
