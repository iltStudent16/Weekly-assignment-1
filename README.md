# SummitGuard Insurance Website

A modern, responsive multi-page insurance website for **SummitGuard Insurance**, built with HTML5, CSS (custom + Bootstrap 5), and JavaScript.

## Project Structure

```
project-root/
├── index.html
├── quote.html
├── about.html
├── faq.html
├── landing.html
├── css/
│   └── styles.css
├── js/
│   ├── main.js
│   └── quote.js
└── README.md
```

## Branding & Colors
- **Primary Brand:** SummitGuard Insurance
- **Palette:** Simple, modern: black (#222), orange (#ff9800), yellow (#ffe082), blue (Bootstrap blue for accents)

## Features
- Modern Bootstrap 5 UI
- Responsive design
- Dynamic quote builder (auto, home, life)
- Navigation bar and footer on all pages
- Custom hero, cards, testimonials
- Organized CSS/JS
- Simple, bold color palette (black, orange, yellow, blue)
- Option to get another quote and save quote
- Git versioned, ready for deployment

## Usage
Open `landing.html` in your browser to start. Navigate using the menu or buttons.

Go to `quote.html` to get a dynamic insurance quote. All forms are validated and interactive. After getting a quote, you can get another quote or save your quote (saving is a placeholder for now).

## Customization
- Update content in HTML files as needed
- Adjust styles in `css/styles.css`
- Modify logic in `js/main.js` or `js/quote.js`
